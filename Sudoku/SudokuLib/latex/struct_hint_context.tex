\doxysubsection{Detailed Description}
Snapshot and precomputed candidate context for hint strategies. 

\doxylink{struct_hint_context}{Hint\+Context} computes candidate lists for every empty cell once and provides read-\/only access to the \doxylink{class_i_sudoku_game}{ISudoku\+Game} required by strategies.
