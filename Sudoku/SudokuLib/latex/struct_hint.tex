\doxysubsection{Detailed Description}
Representation of a hint returned by hint providers/strategies. 

A \doxylink{struct_hint}{Hint} describes a target cell (row/col), an optional suggested value, the hint type, an optional list of candidate digits and a human-\/readable message suitable for UI display.
