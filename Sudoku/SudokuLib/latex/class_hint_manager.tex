\doxysubsection{Detailed Description}
\doxylink{class_hint_manager}{Hint\+Manager} aggregates strategies and selects a hint to return. 

\doxylink{class_hint_manager}{Hint\+Manager} builds a \doxylink{struct_hint_context}{Hint\+Context} (precomputes candidates), queries all configured strategies, and selects one of the returned hints using a weighted random selection based on \doxylink{struct_hint_manager_1_1_strategy_entry_a06b8d4887376cc8e7b39cd08fb831fe5}{Strategy\+Entry\+::weight}.
