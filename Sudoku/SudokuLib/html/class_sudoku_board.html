Detailed Description

Represents the Sudoku board state including mutable cells, initial cells and the solution.

SudokuBoard provides accessor and mutator functions for a 9x9 Sudoku grid, tracks which cells are part of the initial puzzle (fixed) and stores the full solution separately.
