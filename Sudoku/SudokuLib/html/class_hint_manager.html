Detailed Description

HintManager aggregates strategies and selects a hint to return.

HintManager builds a HintContext (precomputes candidates), queries all configured strategies, and selects one of the returned hints using a weighted random selection based on StrategyEntry::weight.

Request the next hint for the provided game state.

The implementation builds a HintContext, invokes each strategy and collects all produced hints. Then a weighted random selection chooses which hint to return. If no strategy returns a hint, std::nullopt is returned.
