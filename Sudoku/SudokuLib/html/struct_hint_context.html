Detailed Description

Snapshot and precomputed candidate context for hint strategies.

HintContext computes candidate lists for every empty cell once and provides read-only access to the ISudokuGame required by strategies.

Compute candidate lists for all cells and store them in candidates.

This examines each empty cell and populates its candidates vector with digits 1-9 that pass isSafeCandidate(). Non-empty cells receive an empty list.

◆ candidates

std::vector<std::vector<int> > HintContext::candidates

Precomputed candidate lists for each cell.

This is a vector of length 81 where index = row*9 + col. Empty cells contain a vector of possible candidates (1-9). Non-empty cells have an empty candidate vector.
